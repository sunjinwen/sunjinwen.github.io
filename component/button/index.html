# Button 按钮

常用的操作按钮。

友情提示

目前icon图标库没维护，暂时icon不能用，后续会陆续添加

# 基础用法

基础的按钮用法。

点击查看代码 
<tms-button>默认按钮</tms-button>
<tms-button type="primary">主要按钮</tms-button>

<tms-button round>圆角按钮</tms-button>
<tms-button type="primary" round>主要按钮</tms-button>

<tms-button icon="ivu-icon ivu-icon-md-alarm" circle></tms-button>
<tms-button type="primary" icon="ivu-icon ivu-icon-md-alarm" circle></tms-button>

# 禁用状态

按钮不可用状态。

点击查看代码 
<tms-button disabled>默认按钮</tms-button>
<tms-button type="primary" disabled>主要按钮</tms-button>

# 图标按钮

带图标的按钮可增强辨识度（有文字）或节省空间（无文字）。

点击查看代码 
<tms-button type="primary" icon="ivu-icon ivu-icon-md-alarm"></tms-button>
<tms-button type="primary" icon="ivu-icon ivu-icon-md-alarm">主要按钮</tms-button>

# 加载中

点击按钮后进行数据加载操作，在按钮上显示加载状态。

点击查看代码 
<tms-button type="primary" loading>加载中</tms-button>

# 不同尺寸

Button 组件提供除了默认值以外的三种尺寸，可以在不同场景下选择合适的按钮尺寸。

点击查看代码 
<tms-button>默认按钮</tms-button>
<tms-button type="primary">主要按钮</tms-button>

<tms-button size="small" round>默认按钮</tms-button>
<tms-button size="small" type="primary" round>主要按钮</tms-button>

# Attributes

参数 说明 类型 可选值 默认值
size 尺寸 string small -
type 类型 string primary -
round 是否圆角按钮 boolean - false
circle 是否圆形按钮 boolean - false
loading 是否加载中状态 boolean - false
disabled 是否禁用状态 boolean - false
icon 图标类名 string - -
autofocus 是否默认聚焦 boolean - false
nativeType 原生 type 属性 string button / submit / reset button
