# ListPagetable 列表页table

用于列表页面table的配置，自动生成所需table

# 基础用法

<tms-list-page-table></tms-list-page-table>

# Attributes

参数 说明 类型 可选值 默认值
gridHeadName 前后端定义的字段 string - -
customColumns 自定义的数据，必须保持和公共表头中的数据格式一致，字段一致，请自行转化 array - -
fetchList 获取表格数据的接口 function(resolve)，resolve 用于赋值 - -
hasCheckboxColumn 是否有选择列 boolean - true
hasOperateColumn 是否有操作列 boolean - true
columnConfig 每一列的配置选项，具体见下表 object - -

# columnConfig

参数 说明 类型 可选值 默认值
[key] 每一列的key，具体见下表 object - -

# [key]

参数 说明 类型 可选值 默认值
type 每一列的筛选项类型，每个筛选项已内置属性，如需要可覆盖 text / textarea / number / date / select / city - text
filterWidget 每一列筛选项配置，属性和对应的组件props一致，emit方法已内置，暂不支持主动调用 object - -
columnWidget 每一列显示配置，跟vxe-grid下的columns属性一致，链接 (opens new window) object - -

type可选值对应的tms组件

  • text / textarea / number —— tms-input（Input输入框）
  • date —— tms-date-picker（日期选择器）默认设置： type: datetimerange valueFormat: yyyy-MM-dd HH:mm:ss format: yyyy-MM-dd HH:mm:ss clearable: false shortcut: true
  • select —— tms-select（Select选择器）默认设置：multiple: true multipleShowTag: false placeholder: ''
  • city —— tms-input（Input输入框）默认设置：placeholder: ''

columnWidget注意点：
针对slots下的default，组件内部已做处理，针对所有类型row中会增加一个 transferShowLabel 字段，该字段为转化后的值（例如：select已转化成对应的label返回出去，使用者可不做转化了，可直接使用），主要用于不同的样式展示

# Events

事件名称 说明 回调参数

on-checkbox-change checkbox变化时触发 选中的row组成的数组
on-table-btn-click 操作列按钮点击时触发 btn - 按钮信息, row - 当前行信息
